Base64 é um algoritmo de codificação que permite transformar qualquer caractere de qualquer idioma em um alfabeto composto por letras, dígitos e sinais latinos. É frequentemente usado para transmitir dados binários por meio de transmissões somente de texto, por exemplo, para enviar imagens e arquivos anexados por e-mail. O método Base 64 consiste em 64 caracteres, sendo [A-Za-z0-9, / e +.

Cifra de César é uma das técnicas de criptografia mais simples e conhecidas. É um tipo de cifra de substituição em que cada letra do texto é substituída por outra. O método recebeu o nome de Júlio César, que o usava para se comunicar com seus generais.

A transformação pode ser representada pelo alinhamento de dois alfabetos. O alfabeto cifrado é o alfabeto normal girado para a direita ou para a esquerda por um número de posições. Para criptografar uma mensagem, basta olhar para cada letra da mensagem na linha Normal e escrever a letra correspondente na linha Criptografar. Para descriptografar, faça o contrário.
